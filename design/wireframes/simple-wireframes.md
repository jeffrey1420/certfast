Requirements with uploaded evidence show a checkmark and file count, creating clear visual feedback. Progress tracking at the bottom shows "2 of 5 requirements complete (40%)" with action buttons to save work or submit for review, guiding users toward task completion.

## Design Notes

### Navigation Patterns
- **Landing Page**: Simple top navigation with clear conversion paths
- **Dashboard & Detail Pages**: Persistent sidebar for quick section access
- **Contextual Navigation**: Back buttons on detail pages for easy return

### Visual Hierarchy
1. **Headers**: Consistent logo placement and user controls
2. **Primary Actions**: High-contrast buttons for CTAs and main workflows
3. **Status Indicators**: Clear badges and progress indicators for state visibility
4. **Secondary Info**: Metadata and timestamps in subdued styling

### Key Interactions
- Drag-and-drop file upload with visual feedback
- Checklist completion with progress tracking
- Metric cards with summary statistics
- Progress bars showing completion percentages
